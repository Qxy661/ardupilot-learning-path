# ArduPilot 调参检查清单

> 系统化调参流程, 打印出来随身携带

---

## 一、起飞前检查

### 硬件检查

- [ ] 螺旋桨安装正确 (CW/CCW 位置)
- [ ] 螺旋桨无裂纹、无变形
- [ ] 电机转向正确 (与混控器匹配)
- [ ] 电调接线正确 (PWM 1-4)
- [ ] GPS 天线朝上, 无遮挡
- [ ] 电池电压正常 (4S: 16.8V 满电)
- [ ] 电池固定牢靠
- [ ] 重心位置合理

### 软件检查

- [ ] 固件版本正确
- [ ] 机架类型选择正确 (X 型四旋翼)
- [ ] 传感器校准完成 (全部绿色)
- [ ] 遥控器校准完成
- [ ] 电调校准完成
- [ ] 飞行模式配置正确
- [ ] 失控保护设置正确 (RTL 或 Land)
- [ ] 电池参数配置正确

### 环境检查

- [ ] 场地空旷 (> 20m × 20m)
- [ ] 无风或微风 (< 3m/s)
- [ ] 无人员在附近
- [ ] GPS 信号良好 (> 10 颗卫星)

## 二、首飞测试

### 悬停测试 (30 秒)

- [ ] 起飞到 2-3m
- [ ] 观察: 无振荡
- [ ] 观察: 位置稳定 (< 0.3m 漂移)
- [ ] 观察: 电机声音平稳
- [ ] 记录: 电流值 (应 < 悬停油门的 1.5 倍)

### 小幅度操控

- [ ] 前后 (±1m)
- [ ] 左右 (±1m)
- [ ] 旋转 (±30°)
- [ ] 观察: 响应正常, 无异常

## 三、角速率环调参

### 滚转轴 (Roll Rate)

参数: `ATC_RAT_RLL_P`, `ATC_RAT_RLL_I`, `ATC_RAT_RLL_D`

| 步骤 | 操作 | 观察 |
|------|------|------|
| 1 | P=0.1, I=0, D=0 | 基准响应 |
| 2 | 增大 P | 振荡? 响应快? |
| 3 | 减小 P 到临界值 | 最大无振荡 P |
| 4 | 加入 D=0.003 | 抑制振荡 |
| 5 | 加入 I=0.135 | 消除稳态误差 |

**判断标准**:
- P 过大: 高频振荡 (10-20Hz), 电机声音尖锐
- P 过小: 响应迟钝, 操控感差
- D 过大: 低频抖动 (1-5Hz)
- I 过大: 低频大幅摆动

### 俯仰轴 (Pitch Rate)

参数: `ATC_RAT_PIT_P`, `ATC_RAT_PIT_I`, `ATC_RAT_PIT_D`

- [ ] 与滚转轴参数相同或接近
- [ ] 测试方法相同

### 偏航轴 (Yaw Rate)

参数: `ATC_RAT_YAW_P`, `ATC_RAT_YAW_I`, `ATC_RAT_YAW_D`

- [ ] 偏航响应正常
- [ ] 无抖动

## 四、姿态环调参

### 滚转轴 (Roll)

参数: `ATC_ANG_RLL_P`

- [ ] 默认值 4.5 通常可接受
- [ ] 阶跃响应: 超调 < 20%
- [ ] 调节时间 < 1s

### 俯仰轴 (Pitch)

参数: `ATC_ANG_PIT_P`

- [ ] 与滚转轴相同

### 偏航轴 (Yaw)

参数: `ATC_ANG_YAW_P`

- [ ] 偏航响应正常
- [ ] 松杆后偏航稳定

---

## 五、位置环调参

### 水平位置

参数: `PSC_POSXY_P`, `PSC_VELXY_P`, `PSC_VELXY_I`, `PSC_VELXY_D`

- [ ] 位置跟踪: 误差 < 0.3m
- [ ] 速度响应: 无振荡
- [ ] 松杆后: 位置保持稳定

### 垂直位置

参数: `PSC_POSZ_P`, `PSC_VELZ_P`, `PSC_VELZ_I`

- [ ] 高度保持: 误差 < 0.2m
- [ ] 阶跃响应: 超调 < 20%
- [ ] 松杆后: 高度稳定

---

## 六、推力模型

参数: `MOT_THST_EXPO`, `MOT_THST_HOVER`

- [ ] 悬停油门设置正确
- [ ] 推力线性度可接受
- [ ] 最低油门足够维持姿态

## 七、Autotune 验证

- [ ] 无风环境下进行
- [ ] 电池电量充足 (> 50%)
- [ ] 场地空旷 (> 20m × 20m)
- [ ] 随时准备切换手动模式
- [ ] Autotune 后手动验证

## 八、安全测试

### 失控保护

- [ ] 遥控器关机 → 触发 RTL
- [ ] GPS 丢失 → 切换到 AltHold
- [ ] 电池低压 → 触发降落
- [ ] 通信丢失 → 触发 RTL

### 紧急操作

- [ ] 手动切换模式正常
- [ ] Kill switch 工作正常
- [ ] 紧急解锁/上锁正常

## 九、日志分析

### 关键指标

| 指标 | 正常范围 | 异常处理 |
|------|----------|----------|
| Roll/Pitch | ±5° (悬停) | 调整姿态环 P |
| 角速率跟踪 | 跟踪误差 < 20°/s | 调整角速率环 P |
| 位置误差 | < 0.3m | 调整位置环 P |
| 振动 | < 3 m/s² | 检查螺旋桨/电机 |
| GPS HDOP | < 2.0 | 更换 GPS 模块 |
| 电池电压 | > 14V (4S) | 更换电池 |
